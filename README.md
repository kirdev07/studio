# KirDev Studio

KirDev Studio — это независимая хобби-студия. Мы разрабатываем мобильные приложения под Android, ботов для социальных сетей и веб-сайты. Весь наш софт (проекты) делается ради интереса и распространяется абсолютно бесплатно.

## Наши направления

- **Android-приложения:** Полезные утилиты и сервисы для мобильных устройств.
- **Чат-боты:** Умные боты для ВКонтакте и Telegram с поддержкой мощного движка BotBrain.
- **Сайты:** Современные, минималистичные и быстрые веб-интерфейсы.
